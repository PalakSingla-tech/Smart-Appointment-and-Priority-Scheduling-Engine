# Smart-Appointment-and-Priority-Scheduling-Engine
A priority-based appointment management engine that schedules and manages appointments based on person priority levels and slot availability.

# Key Features
1. Priority-Based Booking
2. Priority Conflict Detection & Resolution
3. Safe Cancellation of Lower-Priority Appointments
4. Conflict Detection & Slot Validation
5. Concurrent Booking Simulation
6. Modular Service-Based Architecture
7. Clean Console-Based UI

# Tech Stack
1. Java
2. Hibernate ORM
3. MySQL
4. Multithreading
5. Maven
